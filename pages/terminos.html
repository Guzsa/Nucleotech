Al navegar por esta plataforma, el usuario acepta de manera íntegra las condiciones aquí descritas. 2. Naturaleza del Sitio (Proyecto Educativo) Se informa al usuario que Nucleotech es un entorno de simulación desarrollado con fines estrictamente académicos. Los productos, precios y ofertas visualizados forman parte de un ejercicio de programación. En consecuencia, ninguna interacción en el sitio web constituye una transacción comercial real ni genera una obligación de envío o cobro de productos físicos. 3. Propiedad Intelectual Todo el contenido gráfico, logotipos, nombres comerciales y código fuente son propiedad de Nucleotech o han sido utilizados con fines ilustrativos dentro del marco del proyecto. Queda prohibida la reproducción total o parcial del diseño sin autorización previa. 4. Uso del Carrito y Datos de Simulación El proceso de "Checkout" y la carga de datos en el carrito de compras tienen como único objetivo demostrar la funcionalidad de la lógica de JavaScript. Los datos ingresados durante este proceso no serán almacenados en bases de datos permanentes ni utilizados para campañas de marketing. 5. Responsabilidad del Usuario El usuario se compromete a realizar un uso lícito de la plataforma, absteniéndose de intentar vulnerar la seguridad del sitio o utilizar la lógica del carrito de manera malintencionada. 6. Privacidad Nucleotech garantiza que no se recopila información personal sensible. Modificaciones Nos reservamos el derecho de actualizar estos términos para adaptarlos a las nuevas funcionalidades que se agreguen al proyecto durante su etapa de desarrollo.
